# Tools BV
List Backlink Grabber

## Note 
change the name of your web list file to 'sites.txt' so that the tools can run without errors.
This tool uses python 3

## How to use
* $ git clone https://github.com/ilmndwntr/bv.git
* $ cd bv
* $ python3 bv.py
